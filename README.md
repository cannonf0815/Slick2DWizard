# Slick2DWizard
Tutorial for Slick2D Java Game Engine

Targeted for Windows, but easily adaptable for other plattforms.

Based on the Tutorial:
Using Slick 2D to write a game
https://thejavablog.wordpress.com/2008/06/08/using-slick-2d-to-write-a-game/
by Panos Konstantinidis, London - U.K., software engineer, a Java 
Champion and co-founder and co-admin of the hellenic jug 
(http://www.jhug.gr/) in Athens - Greece
a Java Hellenic User Group.

Tools used:
Tiled map editior
http://www.mapeditor.org/
to crate orthogonal map with a 10×10 dimensions
Tiles and sprite size is 32x32

To compile and run use:
> compilenrun.bat

Some usefull resources:
http://slick.ninjacave.com/wiki/index.php
